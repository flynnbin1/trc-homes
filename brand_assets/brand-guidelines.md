# TRC Homes — Brand Guidelines

## Colour Palette

- **Primary: Forest Green `#1E3A2F`** — wordmark, headlines, reversed backgrounds
- **Accent: Muted Gold `#B08D57`** — keystone accents, fine rules — use sparingly
- **Canvas: Off-White `#F7F5EF`** — default background
- **Support: Stone `#E9E5DA`** — secondary panels, cards, quiet section breaks
- **Depth: Deep Pine `#152B22`** — footers, photo overlays, dark surfaces

## Colour Balance

Off-White 55% / Stone 15% / Forest Green 22% / Deep Pine 5% / Gold 3%
